## Propuesta de como organizar las carpetas de un proyecto en vue

```
src/
+-- modules/
|   +-- moduleA/
|   |   +-- componentes/
|   |   +-- services/
|   |   +-- helpers/
|   |   +-- router/
|   |   +-- store/
|   +-- moduleB/
|   +-- moduleC/

```

Intall vue router
```
npm install vue-router@4
```

ubucacion del router de la aplicacion
```
src/

```
